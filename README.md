# Innovative Snake Game

A modern version of the classic Snake Game developed in **Python** using **Pygame**. This project includes local multiplayer, obstacles, power-ups, multiple themes, snake customization, and a simple AI Coach that provides gameplay tips.

## Features

* Local Multiplayer (2 Players)
* Obstacles
* Speed Boost Power-up
* Multiple Environment Themes
* Snake Color Customization
* AI Coach Tips
* Score Tracking
* Game Over & Restart Screen

## Controls

### Player 1

* **W** - Up
* **A** - Left
* **S** - Down
* **D** - Right

### Player 2

* **↑** - Up
* **←** - Left
* **↓** - Down
* **→** - Right

### Menu

* **Enter** - Start Game
* **T** - Change Theme
* **C** - Change Snake Color

## Installation

```bash
pip install -r requirements.txt
python game.py
```
